# SuprMentr Full Stack Web Development Internship

This repository is my working log for the **SuprMentr Full Stack Web Development Internship**. It collects assignment submissions, mini projects, API practice, and supporting notes in one place so the full progression is easy to review.

## What This Repository Contains

- Date-wise internship assignments from `Assignment-1` through `Assignment-25`
- A `Self-Assignment` folder for independent practice
- Frontend practice projects built with HTML, CSS, JavaScript, React, and Vite
- Backend practice projects built with Node.js, Express.js, and MVC patterns
- Supporting instruction files such as `Assignment_1.txt`, `Assignment_15.txt`, and similar task briefs

## Repository Layout

The workspace is organized into two broad types of work:

1. Assignment folders
	- Progression-based work arranged by date
	- Includes both frontend and backend exercises
	- Used to document learning across the internship timeline

2. Task folders
	- Focused practice projects with a specific goal
	- Covers UI validation, list management, search, blog, e-commerce, student APIs, product APIs, and MVC architecture

## Tech Stack

- **Frontend:** HTML, CSS, JavaScript, React, Vite
- **Backend:** Node.js, Express.js
- **Patterns:** MVC, REST-style APIs, component-based UI

## How To Use This Repo

- Open the folder for the assignment or task you want to review
- Read the matching instruction file when present
- Run the relevant frontend app with Vite or backend app with Node/Express, depending on the folder
- Use the repository as a reference for assignment history and practical implementation examples
